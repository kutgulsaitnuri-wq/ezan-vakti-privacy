Gizlilik Politikasi

Bu gizlilik politikasi, Ezan Vakti mobil uygulamasinin ("Uygulama") kullanici verilerini nasil isledigini aciklamaktadir. Uygulamayi kullanarak bu politikayi kabul etmis sayilirsiniz.

1. Toplanan Veriler

1.1 Konum Verisi

Uygulama, namaz vakitlerini ve kible yonunu dogru hesaplayabilmek icin cihazinizin konum bilgisine (GPS) erisim izni talep eder. Konum verisi:

  • Yalnizca namaz vakti hesaplama ve kible yonu belirleme amacli kullanilir
  • Yalnizca cihazinizda yerel olarak saklanir
  • Hicbir sunucuya veya ucuncu tarafa gonderilmez

1.2 Yerel Depolama

Uygulama, tercihlerinizi cihazinizda yerel olarak saklar. Bunlar:

  • Tema tercihi (karanlik/aydinlik mod)
  • Bildirim ayarlari
  • Son kullanilan konum bilgisi
  • Zikirmatik sayac degerleri
  • Favori hadisler
  • Hesaplama yontemi ve ezan sesi tercihi
Bu veriler yalnizca cihazinizda saklanir ve hicbir dis sunucuya aktarilmaz.

2. Ucuncu Taraf Hizmetleri

Uygulama, asagidaki ucuncu taraf API'lerini kullanir:

  • Aladhan API - Namaz vakitlerini hesaplamak icin konum koordinatlari (enlem/boylam) gonderilir
  • Al-Quran Cloud API - Kuran ayetlerini getirmek icin sure numarasi gonderilir

Bu hizmetlere yalnizca islevsel olarak gerekli olan minimum veri gonderilir. Kisisel bilgi paylasimi yapilmaz.

3. Toplanmayan Veriler

Uygulama asagidaki verileri toplamaz, saklamaz veya paylasmaz:

  • Kisisel kimlik bilgileri (ad, e-posta, telefon numarasi vb.)
  • Hesap veya giris bilgileri
  • Kullanim analitigi veya izleme verileri
  • Reklam kimlikleri
  • Finansal veya odeme bilgileri

4. Veri Guvenligi

Tum kullanici tercihleri cihazinizda yerel olarak saklanir. Uygulama herhangi bir kullanici verisi icin sunucu tarafli depolama kullanmaz. API istekleri HTTPS protokolu uzerinden guvenlibir sekilde gerceklestirilir.

5. Cocuklarin Gizliligi

Uygulama kisisel veri toplamadigi icin cocuklar dahil tum yas gruplari tarafindan guvenle kullanilabilir. 13 yas altindaki cocuklardan bilerek kisisel bilgi toplanmaz.

6. Izinler

  • Internet Erisimi - Namaz vakitleri ve Kuran ayetleri icin API baglantisi
  • Konum Erisimi - Namaz vakti ve kible yonu hesaplamasi (kullanici izni ile)

7. Degisiklikler

Bu gizlilik politikasi zaman zaman guncellenebilir. Degisiklikler bu sayfada yayinlanacaktir. Uygulamayi kullanmaya devam etmeniz, guncellenmis politikayi kabul ettiginiz anlamina gelir.
